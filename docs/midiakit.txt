FUNCIONALIDADES PRINCIPAIS
================================================================================

1. EXTRAÇÃO AUTOMÁTICA DE TELEFONES
   - Sistema identifica e extrai números de telefone automaticamente
   - Adiciona botão WhatsApp instantâneo para contato rápido
   - Nunca mais perca um número de telefone importante
   - Tudo salvo automaticamente no banco de dados

2. AGENTE IA INTELIGENTE
   - Solicita telefone automaticamente quando cliente não forneceu
   - Gera mensagens personalizadas e profissionais
   - Analisa respostas do cliente para extrair informações
   - Detecta se cliente prefere ligação ou WhatsApp
   - Envia mensagens de agradecimento automaticamente
   - Funciona mesmo quando você não está online

3. PROCESSAMENTO AUTOMÁTICO DE CONVERSAS
   - Detecta e processa novas conversas automaticamente
   - Identifica conversas não lidas e abre automaticamente
   - Extrai todas as mensagens e salva no banco de dados
   - Classifica leads automaticamente (lead ou não-lead)
   - Extrai URLs de propriedades das mensagens

4. BANCO DE DADOS CENTRALIZADO
   - Todas as conversas salvas automaticamente
   - Histórico completo de mensagens
   - Dados protegidos (números nunca são sobrescritos)
   - Acesso via dashboard web
   - Relatórios e análises disponíveis

5. MONITORAMENTO EM TEMPO REAL
   - Detecta novas mensagens instantaneamente
   - Processa e salva automaticamente
   - Notifica sobre leads importantes
   - Nunca perca uma oportunidade
